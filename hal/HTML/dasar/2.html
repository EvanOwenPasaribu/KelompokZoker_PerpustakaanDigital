HTML Dasar part 2 - Element HTML

Elemen HTML merupakan komponen yang menetapkan peran sebuah objek dalam dokumen, termasuk struktur dan konten dari objek tersebut.
Contoh :
<h1>,<p>, <b>, <i>
<span>,<div>, <table>
<section>,<article>, <header>

Akan ada beberapa elemen yang akan saya jelaskan di bawah ini.

1. Heading

ada 6 element heading di dalam HTML
<h1>,<h2>,<h3>,<h4>,<h5>,<h6>

<!DOCTYPE html>
<html>
<head>
     <title>Heading</title>
</head>
<body>
     <h1>Heading pertama</h1>
     <h2>Heading kedua</h2>
     <h3>Heading ketiga</h3>
     <h4>Heading keempat</h4>
     <h5>Heading kelima</h5>
     <h6>Heading keenam</h6>
</body>
</html>

2. Paragraf

Untuk membuat paragraf pada halaman web, biasanya menggunakan elemen <p>

<!DOCTYPE html>
<html>
<head>
     <title>Paragraf</title>
</head>
<body>
     <h1>Paragraf</h1>
     <p>Ini adalah sebuah paragraf, main menu paragraf biasanya berada pada akhir paragraf atau di akhir paragraf</p>
</body>
</html>

Hasil

3. Atribut Align

Atribut align mengindikasikan apakah tulisan disejajarkan ke kiri, tengah, atau ke kanan pada halaman web

<!DOCTYPE html>
<html>
<head>
     <title>Atribut Align</title>
</head>
<body>
     <h1 align = "center">Align</h1>
     <p align = "left">Rata Kiri</p>
     <p align = "center">Rata Tengah</p>
     <p align = "right">Rata Kanan</p>
</body>
</html>

Hasil

4. Bold, Italic dan Underline

elemen ini diletakkan di dalam elemen lain.
<b> = membuat teks menjadi lebih tebal.
<i> = membuat teks menjadi miring.
<u> = membuat garis bawah pada teks.

<!DOCTYPE html>
<html>
<head>
     <title>Bold,Italic,Underline</title>
</head>
<body>
     <h1>Bold,Italic,Underline </h1>
     <p>Membuat Teks Cetak <b>Tebal</b> Pada Website</p>
     <p>Membuat Teks Cetak <i>Miring</i> Pada Website</p>
     <p>Membuat Teks Dengan <u>Garing Bawah</u> Pada Website</p>
</body>
</html>

Hasil

5. Superskrip dan Subskrip

elemen ini diletakkan di dalam elemen lain.
<sup> = membuat teks menjadi lebih kecil dan berada di atas.
<sub> = membuat teks menjadi lebih kecil dan berada di bawah.

<!DOCTYPE html>
<html>
<head>
     <title>Superskrip dan Subskrip</title>
</head>
<body>
     <h1>Superskrip dan Subskrip</h1>
     <p>2<sup>10</sup> sama dengan 1024</p>
     <p>O<sub>2</sub> adalah simbol kimia untuk oksigen</p>
</body>
</html>

Hasil

6. Elemen s dan strike

elemen ini diletakkan di dalam elemen lain.
<s> dan <strike> = Menampilkan Teks yang di coret.

<!DOCTYPE html>
<html>
<head>
     <title>Elemen s dan strike</title>
</head>
<body>
     <h1>Elemen s dan strike</h1>
     <p>kata ini akan <s>dicoret</s> </p>
     <p>kata ini akan <strike>dicoret</strike> juga</p>
</body>
</html>

Hasil

7. Elemen big, small, dan strong

elemen ini diletakkan di dalam elemen lain.
<big> = Membuat teks menjadi lebih besar.
<small> = Membuat teks menjadi lebih kecil.
<strong> = Untuk menunjukkan penekanan kuat pada kontennya.

<!DOCTYPE html>
<html>
<head>
     <title>Elemen big, small, dan strong</title>
</head>
<body>
     <h1>Elemen big, small, dan strong</h1>
     <p>kata ini akan ditampikan <big>lebih besar</big> dari kata lain</p>
     <p>kata ini akan ditampikan <small>lebih kecil</small> dari kata lain</p>
     <p>kata ini akan ditampikan <strong>lebih tebal</strong> dari kata lain</p>
</body>
</html>
